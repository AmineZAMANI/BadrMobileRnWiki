Préparer l'environnement

  1. Allez sur https://nodejs.org/en/download/ et installez NodeJs.
> node -v
  1. Installer React Native CLI
> npm install –g react-native-cli
  1. Créez un nouveau projet
> react-native init Counter001
  1. Allez sur le répertoire du projet
> cd Counter001
  1. Démarrer l'application
> react-native run-android

  1. Installez l'outil VisualCode puis ouvrez le dossier du projet.

  2. Installez le plugin React native Tools
