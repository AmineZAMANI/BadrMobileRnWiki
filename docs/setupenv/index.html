Préparer l'environnement

  1. Téléchargez git depuis : https://git-scm.com/downloads puis récuperer les sources depuis github via la commande :
> git clone https://github.com/AmineZAMANI/BadrMobileRn.git

Installez l'outil VisualCode puis ouvrez le dossier du projet.

  1. Installez le plugin React native Tools

  1. Allez sur https://nodejs.org/en/download/ puis installez la dernière version de NodeJs.

> node -v

  1. Allez sur https://developer.android.com/studio puis installez la dernière version d'Android studio avant de vérifier qu'au moins un sdk platform est bien installé (Exemple : API 29) :

  2. Installez React Native CLI

> npm install –g react-native-cli
  1. Ouvrez le projet que vous avez récupéré depuis git dans Visual Code puis dans un nouveau terminal lancez la commande :
> npm install

  1. Ouvrez Android studio, créez un nouveau virtual device puis lancez le en cliquant sur "Start" :

  2. Démarrez l'application via la commande :

> react-native run-android

  1. Dés que l'application est lancée vérifiez que le hot reloading marche bien en s'assurant que l'écran s'actualise lors d'un CTRL+S éxécuté sur une vue :
